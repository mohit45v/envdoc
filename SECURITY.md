# Security Policy

## Supported Versions

Currently, only the latest version of this project is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| > 1.0.x | :white_check_mark: |
| < 1.0.x | :x:                |

## Reporting a Vulnerability

We take the security of this project very seriously. If you discover a security vulnerability in `envdoc-ai`, please do NOT open a public issue.

Instead, please send an email to the repository owner directly. You can expect a response acknowledging the vulnerability within 48 hours.

Once the vulnerability is confirmed, we will work to patch it and release an update as quickly as possible. We will also credit you for the responsible disclosure if you desire.
